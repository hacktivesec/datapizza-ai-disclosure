This repository contains public information for the disclosure of two vulnerabilities in [datapizza-ai](https://github.com/datapizza-labs/datapizza-ai) framework:

- **CVE-2026-2969**: Server Side Template Injection in datapizza-ai prompt evaluation ([Read details here](https://github.com/hacktivesec/datapizza-ai-disclosure/blob/main/ssti.md))
- **CVE-2026-2970**: Unsafe Deserialization in datapizza-ai Redis cache ([Read details here](https://github.com/hacktivesec/datapizza-ai-disclosure/blob/main/unsafe-deserialization.md))
